# 🚀 JanPramaan – Civic Transparency Platform

## 📌 Overview

**JanPramaan** is a civic-tech platform that brings **transparency and accountability** to public infrastructure projects.

It allows citizens to verify whether public works are actually completed using **geo-tagged, evidence-based tracking**.

---

## 🎯 Problem

* Lack of transparency in public works
* No real-time verification
* High chances of corruption

---

## 💡 Solution

* View ongoing & completed projects
* Upload geo-tagged proof
* Track project lifecycle
* Ensure data authenticity using location

---

## 📱 Mobile App Features

* 🌐 WebView-based live platform
* 📍 Location access
* 🔄 Refresh support
* 📡 Connectivity detection
* ⏳ Loading indicator
* ⬅️ Back navigation

---

## 🛠️ Tech Stack

* Flutter (Dart)
* WebView Flutter
* Connectivity Plus
* Permission Handler
